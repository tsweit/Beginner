Displaying special characters
Special characters can be characters that cannot be typed with a keyboard for example the trademark and copyright symbols. A special character can also be a symbol that is reserved for use by HTML for example the left and right angle brackets used for tags.
If I intend to show the "<" and ">" without rendering issues as shown above I will have to identify the name character entity for each symbol and add it to the ampersand symbol as shown here as well "™" "©".

Displaying Images
When displaying images, we have to use the "img" tag and tell the browser where to get the image from. We can also change the weight and height of the image.
The src attribute tells the browser where to find the image and the alt attribute allows users to add a description.
